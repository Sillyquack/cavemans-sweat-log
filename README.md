# The Caveman's Sweat Log

Lift. Log. Evolve.

En enkel React/Vite treningslogg for apparater, sets, reps, kg, kroppsvekt, midje og BMI.

## Kjør lokalt

```bash
npm install
npm run dev
```

Åpne lenken som vises i terminalen.

## Bygg for produksjon

```bash
npm run build
npm run preview
```

## GitHub Pages senere

I `vite.config.js`, sett `base` til repo-navnet ditt:

```js
base: '/cavemans-sweat-log/'
```

Så kan vi legge inn deploy-oppsett når du er klar.
